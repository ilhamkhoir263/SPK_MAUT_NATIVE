# 📊 MAUT System: Fruit Quality Intelligence

An automated **Decision Support System (DSS)** implementing the **Multi-Attribute Utility Theory (MAUT)**. This platform transforms subjective fruit grading into data-driven results through mathematical utility modeling.

---

### 💡 Core Logic & Algorithm
The system processes data through four critical stages to ensure precision:
1. **Weighting**: Assigning importance levels to each quality criterion.
2. **Normalization**: Converting raw criteria values into a standard utility scale.
3. **Utility Calculation**: Computing the final utility value for each alternative.
4. **Ranking**: Sorting alternatives based on scientific utility scores.

### 🛠️ Technical Breakdown
| Component | Technology |

### ⚡ Key Features
* **Multi-Criteria Evaluation**: Evaluate alternatives based on various weighted attributes.
* **Real-time Scoring**: Instant utility score generation upon data entry.
* **Automated Decision Table**: View all calculated values in a structured, transparent table.
